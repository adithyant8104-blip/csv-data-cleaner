# CSV Data Cleaner & Analyzer

A beginner-friendly data science project demonstrating real-world data cleaning,
exploratory analysis, and visualisation using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## What This Project Does
- Loads a messy CSV dataset
- Handles missing values and duplicates
- Performs basic statistical analysis
- Visualises price distribution
- Exports a cleaned dataset

## Outputs
- Cleaned CSV file
- Histogram plot

This project is part of my MSc Advanced Data Science & AI learning journey.
